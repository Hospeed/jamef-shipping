Jamef_Shipping
==============

> Módulo de frete da Transportadora Jamef para Magento


## Testado em Magento

`1.6.2.0`, `1.7.0.2`, `1.8.1.0` e `1.9.0.1`

## Instalando manualmente

Copie a pasta app para a raiz do seu Magento
